# Financial-Performance-Analysis-Dashboard
Interactive financial analytics dashboard visualizing revenue, profitability, and macroeconomic trends across various companies using Power BI.
A Power BI dashboard analyzing financial performance across multiple companies, featuring KPI cards for average revenue, net income, expenses, and profit margin. Includes quarterly trend analysis, interest rate vs. profit margin scatter plots, high/low profitability comparisons, and inflation rate overlays — built to surface seasonal patterns and identify outlier firms.
